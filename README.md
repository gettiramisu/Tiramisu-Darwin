## Installation
1. Download the latest `dmg` from the [releases](https://github.com/gettiramisu/Tiramisu-Darwin/releases)
2. Drag and drop the `app` into the `Applications` directory

## Compiling
### macOS
1. Install the dependencies:
    - `Xcode`: 14 or later
2. Clone the project: `git clone https://github.com/gettiramisu/Tiramisu-Darwin.git && cd ./Tiramisu-Darwin/ && open ./App.xcworkspace`
3. Build and run the project
